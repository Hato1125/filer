# Bundled Fonts

The Material Symbols font files in this directory are derived from Google's
Material Symbols variable fonts.

Source:

- https://fonts.google.com/icons
- https://github.com/google/material-design-icons

License:

- Apache License 2.0
- See `LICENSE-APACHE-2.0.txt`.

Generated files:

- `MaterialSymbolsFilled.ttf`
  - Source: `MaterialSymbolsOutlined[FILL,GRAD,opsz,wght].ttf`
  - Axes: `FILL=1 GRAD=0 opsz=24 wght=400`
- `MaterialSymbolsRounded.ttf`
  - Source: `MaterialSymbolsRounded[FILL,GRAD,opsz,wght].ttf`
  - Axes: `FILL=0 GRAD=0 opsz=24 wght=400`
